Features

Some of our key platform features and capabilities to help our customers stay ahead of market trend.

Folders

Create an in-tenant file storage or link a cloud storage such as Microsoft OneDrive or Google Drive. Folders can be shared to other team members who has access to the tenant portal.

Scheduler

A user defined job scheduler, define the occurrences to run a flow of actions, such as sending email, calling a web service, or trigger a workflow to run.

Calendar View

Calendar View is an UI for listing or viewing of documents in calendar view of Document Collections, supporting daily, weekly, and monthly views.

Workflow Designer

Drag and drop workflow designer features business process flow with state management.

Dynamic Collections

Define collections for structured and/or unstructured business documents. Relationships between collections can be defined for normalized structure.

Page Designer

Create web pages with drag and drop page designer, CSS and JavaScript editor for customization. Supports widgets, templates, layouts, and form components.

API Designer

Define, design, and create API endpoints without code. Supports GET, POST, PUT, etc., with customizable parameters, content types, and outputs.

AI Bot Assisted Builder

Automatically create applications with AI assistance, generating forms, API endpoints, and other modules via dynamic JSON patterns.
